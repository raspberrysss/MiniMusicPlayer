# MiniMusicPlayer
基于VS，SqlServer采用c#语言开发的简易音乐播放器，可实现不同用户导入本地音乐播放，对不同音乐进行评价反馈，申请成为音乐人，注销账号等功能
播放部分参考b站这个视频https://www.bilibili.com/video/BV1Ds4y1s7ja/?spm_id_from=333.337.search-card.all.click
# 基本功能
1. 用户登录注册
2. 播放音乐
3. 评论音乐
4. 申请注销
5. 申请成为音乐人
6. 管理员删除用户
7. 管理员审核评论

1. 连接SqlServer数据库及操作数据
